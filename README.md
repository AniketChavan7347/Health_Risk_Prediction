# 🩺 Health Risk Predictor

A Machine Learning-based web application that predicts an individual's health risk level (Low, Medium, High) based on medical and lifestyle parameters.

---

## 🚀 Project Overview
This project uses classification algorithms to analyze healthcare data and determine a person’s risk category. The model is deployed using Streamlit to provide real-time predictions through an interactive user interface.

---

## 🛠 Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Joblib
- Streamlit
- Plotly

---

## 📊 Model Performance
- Accuracy: ~97%
- High precision and recall
- Multi-class classification (Low, Medium, High)

---

## 🌐 Deployment
The application is deployed using Streamlit Cloud and can be accessed online.

---

## 🎯 Objective
To demonstrate how Machine Learning can assist in early health risk assessment and preventive healthcare.

---

Developed as a Final Year Data Science Project.
